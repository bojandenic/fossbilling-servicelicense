# Add Servicelicense module for WordPress plugin licensing and secure distribution

This PR adds the Servicelicense module to the FossBilling extension directory.

Servicelicense is a WordPress plugin licensing and distribution module designed for vendors who need:

- Automatic license key generation
- Secure ZIP file distribution
- Product-based configuration
- Version and changelog display
- Domain-bound validation
- License reset functionality

The module uses a secure client controller for downloads and does not expose direct file paths.
